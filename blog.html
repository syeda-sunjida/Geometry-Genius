1. What is DOM?

DOM(Document Object Model) is a kind of platform that consider total HTML as a tree. it can access each and every node of the tree where nodes are considered as the element of the HTML code. With dom , we can dominate that means change, update, add, delete any new element in website. Also we can make a website more iteractive with DOM.

2. How to select element by using DOM?

In 5 ways, we can select elements by using DOM

  1. getElementsByTagName()
  2. getElementsByClassName()
  3. getElementById()
  4. querySelector()
  5. querySelectorAll()

3. What is Event Bubbling?

Event bubbling happens when an element receives an event, and that event bubbles up or propagate to its parent and ancestor elements in the DOM tree until it gets to the root element.

4. What is call back funtion?

A callback is a function passed as an argument to another function. This technique allows a function to call another function. A callback function can run after another function has finished.
